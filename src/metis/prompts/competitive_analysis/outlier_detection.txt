{metric_data}

**Statistical Analysis Requirements:**

1. **Outlier Detection Methods**
   - Z-score analysis (>3 standard deviations)
   - Interquartile Range (IQR) method (1.5 * IQR)
   - Modified Z-score using median absolute deviation
   - Domain-specific outlier rules for insurance metrics

2. **Data Quality Assessment**
   - Missing data identification
   - Impossible value detection (negative P/E, >100% growth rates)
   - Temporal consistency checks
   - Cross-metric validation

3. **Business Logic Validation**
   - Combined ratios <70% or >130% (unusual for insurance)
   - P/E ratios >50x or <5x (potential data errors)
   - ROE >50% or <0% (extreme performance or errors)
   - Market cap consistency with share price calculations

4. **Exclusion Recommendations**
   - Recommend data points to exclude from averages
   - Suggest alternative data sources for flagged metrics
   - Provide confidence scores for remaining data
