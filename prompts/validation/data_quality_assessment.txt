**Quality Assessment Criteria:**

1. **Data Completeness**
   - Required fields presence (>95% complete)
   - Missing data identification and impact
   - Data coverage across time periods
   - Peer group data consistency

2. **Data Accuracy**
   - Cross-validation against multiple sources
   - Mathematical consistency checks
   - Range validation for financial metrics
   - Temporal consistency verification

3. **Data Freshness**
   - Financial data: <90 days old
   - Transcript data: <120 days old
   - Market data: <24 hours old
   - Peer data: Same reporting periods

4. **Data Reliability**
   - Source credibility assessment
   - Data provider track record
   - Audit trail completeness
   - Error rate measurement

**Quality Scoring (0-100 scale):**
- 90-100: High quality, suitable for analysis
- 70-89: Good quality, minor limitations noted
- 50-69: Moderate quality, significant limitations
- Below 50: Poor quality, analysis not recommended
